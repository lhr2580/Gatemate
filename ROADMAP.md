# GateMate 开发路线图

## 项目愿景

GateMate 致力于成为开发者管理 AI API Key 的首选工具，提供智能路由、成本控制、用量监控等核心功能，帮助开发者最大化 API 利用率并降低成本。

## 当前版本: v2.0.0

### 已完成功能

- ✅ API Key 管理（增删改查）
- ✅ 多供应商支持（OpenAI、DeepSeek、Doubao、Qwen、Claude、Gemini、Groq、Ollama）
- ✅ 三种路由策略（轮询、随机、智能路由）
- ✅ 用量统计与图表展示
- ✅ 调用日志记录与过滤
- ✅ 项目管理与预算控制
- ✅ API Key 加密存储（AES-256-GCM）
- ✅ 系统 Keychain 集成
- ✅ 多主题支持
- ✅ CSV/PDF 报表导出
- ✅ 命令面板（Cmd+K）
- ✅ 通知中心
- ✅ 组件化重构

## 短期目标: v2.1.0（2024 Q3）

### 功能增强

- [ ] **实时监控** - WebSocket 实时推送用量数据

- [ ] **API 测试工具** - 内置请求测试功能
- [ ] **快捷键支持** - 更多键盘快捷键
